


How to run code document
1.Install nodeJS on your machine.
2.Install VS code
3.Open project in VScode
4.Complete github setup in VScode
5.Open terminal and Run “npm init” in terminal
6.Run Npm install –save –dev cypress@10.7.0 to install Cypress
7.To open test runner:    npx cypress open
8.Select homepage in test  runner to run the code
